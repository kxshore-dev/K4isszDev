# 🚀 About Me

I'm a 16-year-old developer learning to build things on the web. My coding journey started unexpectedly during lockdown in 2021 when I discovered Telegram bots. What began as curiosity has become a genuine passion for solving problems through code.

Currently, I'm balancing my Class 12th academics while focusing on **building open-source projects** and exploring what I want to do next.

I'm honest about my strengths — Python, databases, backend logic — and my weaknesses — frontend design. I focus more on building useful products and improving every day through practice and experimentation.
